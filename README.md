# 🎬 Netflix Data Analysis

A comprehensive Data Analysis project that explores Netflix's movies and TV shows dataset using Python. This project focuses on cleaning, analyzing, and visualizing Netflix content to uncover trends, patterns, and insights about the platform's catalog.

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to answer various business and analytical questions, such as:

- Distribution of Movies and TV Shows
- Content release trends over the years
- Most common genres
- Country-wise content production
- Ratings distribution
- Popular directors and actors
- Duration analysis
- Missing data handling

The project demonstrates practical data analysis techniques using Python libraries.

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/Netflix-Data-Analysis.git
```

### 2. Navigate to the project directory

```bash
cd Netflix-Data-Analysis
```

### 3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Open the notebook and run all cells.
